# NeRF Studio for 3D reconstruction
## Overview

This repository contains experimental work and a screenshot from exploring NeRF Studio for neural 3D reconstruction.
The objective of this work was to understand how Neural Radiance Fields (NeRF) represent 3D scenes from multi-view images and to compare the reconstruction quality with traditional photogrammetry-based methods.

## Process
- Ran the NeRF Studio training pipeline on multi-view image datasets extracted from video recordings (frame sampling using ffmpeg)
- Investigated neural radiance field optimisation and convergence behaviour during training
- Analysed reconstructed scenes and intermediate rendering outputs
- Evaluated the impact of input data quality and view coverage on reconstruction performance
- Compared qualitative results with traditional photogrammetry reconstruction approaches
 
## Note
This work is exploratory and focused on understanding the pipeline and reconstruction behaviour rather than building a custom implementation.
